# Multiplayer Tetris with AI

실시간 멀티플레이어 테트리스 게임과 AI 플레이어 구현

## 설치 및 실행

1. 저장소 클론:
```bash
git clone https://github.com/cadwin95/multiplayer-tetris.git
cd multiplayer-tetris
```

2. 의존성 설치:
```bash
npm install
```

3. 개발 서버 실행:
```bash
npm run dev
```
